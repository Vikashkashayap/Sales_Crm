# Onboarding PDF documents

Upload and manage these files from **Settings → Document Management** in the admin CRM.

## Folder layout

| Folder | Contents |
|--------|----------|
| `english/` | English-medium welcome kit PDFs |
| `hindi/` | Hindi-medium welcome kit PDFs |
| `common/` | Shared PDFs (e.g. Student Enrollment Form) |

## English filenames

- Mentorsdaily Promise English.pdf
- Book List English.pdf
- Syllabus Analysis English.pdf
- Welcome Letter English.pdf
- Batch Schedule English.pdf
- Fee Structure English.pdf
